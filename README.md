# Bamboo Eater

![Bamboo Eater Logo](./Shared \(App\)/Resources/Icon.png)

A (Safari) browser extension to fix BambooHR.

## Features

- Adds a direct link to the calendar in the sidebar
- Adds functionality to create a time off from the calendar view
- Fixes various UI issues for better usability

## Installation

If you are using Safari, download the repository and open the `Bamboo Eater.xcodeproj` file in Xcode. From there, you can build and run the extension in Safari.

For other browsers, load the extension files as an unpacked extension.
